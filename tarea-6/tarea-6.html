Tarea 6

Elementos de una Red Local

1. Objetivo General

Conocer, investigar e identificar los elementos que se utilizan para configurar una red local y el acceso a recursos de un equipo en red.

2. Objetivos específicos

  • Utilizando el programa de Packet Tracer de Cisco, simula las siguientes redes de computadora, asignando a las computadoras las IP pertenecientes a la red 192.168.0.0., utilizando la ventana de comandos utiliza la función ping para verificar la comunicación entre cada una de las computadoras. Graba un video de cada red simulada.

  • Elabora por cada red una tabla comparativa de cada elemento de red con las siguientes características:

Cable Ethernet

Un cable Ethernet es un tipo de cable de red utilizado para transmitir datos de forma rápida y confiable entre dispositivos, como computadoras, en una red local (LAN). Se utiliza principalmente para la conexión a Internet y para la comunicación entre dispositivos en una red local, permitiendo una transmisión de datos de alta velocidad y una conexión estable.
