¿Qué es lenguaje JavaScript?

JavaScript es un lenguaje de programación de alto nivel, interpretado y orientado a objetos que se utiliza principalmente en el desarrollo web.

¿Para qué sirve el leguaje JS?

JavaScript es un lenguaje de programación fundamental en el desarrollo web. Gracias a su capacidad para agregar interactividad y dinamismo a las páginas web, JavaScript permite crear experiencias de usuario más atractivas y funcionales. Se utiliza para validar formularios, manipular y actualizar el contenido de una página en tiempo real, realizar solicitudes a servidores para cargar datos adicionales, crear efectos visuales y animaciones, y desarrollar aplicaciones web completas tanto en el lado del cliente como en el lado del servidor. Además, JavaScript facilita la integración con APIs y servicios web, lo que permite la comunicación y el intercambio de datos con otros sistemas. En resumen, JavaScript es esencial para crear experiencias web interactivas, mejorar la usabilidad y proporcionar funcionalidades avanzadas en el mundo digital.

¿A qué se refiere el DOM (Document Object model) en el desarrollo Web?

Document Object Model, abreviado como DOM, es una interfaz de programación estandarizada para la estructuración de documentos HTML y XML. Fue desarrollado y publicado por el Consorcio World Wide Web (W3C). El objetivo del Document Object Model es facilitar al máximo a los programadores el acceso a los componentes de un proyecto web para que puedan añadir, eliminar o editar contenido, atributos y estilos.

¿Cuál es la diferencia entre los archivos HTML, CSS y JS?

HTML (HyperText Markup Language)


HTML es el lenguaje de marcado utilizado para estructurar y organizar el contenido de una página web. Define los elementos y su jerarquía, como encabezados, párrafos, imágenes, enlaces, formularios, etc. HTML proporciona la estructura básica de una página web y se utiliza para describir la semántica y la presentación del contenido

CSS (Cascading Style Sheets)


CSS es un lenguaje de hojas de estilo utilizado para definir el aspecto visual y la presentación de una página web. Se utiliza para establecer el color, la tipografía, el diseño, los tamaños, los márgenes y otros aspectos visuales de los elementos HTML definidos en la página. CSS permite separar la estructura del contenido (HTML) de su apariencia y facilita la creación de diseños consistentes y atractivos en múltiples páginas.

JS (JavaScript)


JavaScript es un lenguaje de programación que se utiliza para agregar interactividad y funcionalidad a las páginas web. Permite crear comportamientos dinámicos, responder a eventos del usuario, manipular y modificar el contenido del DOM, realizar validaciones de formularios, acceder a APIs y servicios web, y mucho más. JavaScript se ejecuta en el lado del cliente, en el navegador web, y se utiliza para realizar acciones y cálculos en tiempo real.

¿Para qué sirve la etiqueta <script>?

La etiqueta <script> se utiliza en HTML para insertar o incrustar código JavaScript en una página web. El código JavaScript contenido dentro de la etiqueta <script> se ejecuta cuando se carga o se procesa la página web por un navegador web.

¿En qué consiste los 3 métodos para cambiar elementos tipo Tag Name, ID y Class Name con JS?

Tag Name


Para cambiar elementos basados en su nombre de etiqueta, puedes utilizar el método getElementsByTagName(). Este método devuelve una lista de todos los elementos con el nombre de etiqueta especificado. Puedes acceder a un elemento específico de la lista utilizando su índice y luego modificar sus propiedades o contenido.

ID


Si deseas cambiar un elemento específico basado en su atributo de ID único, puedes utilizar el método getElementById(). Este método devuelve el elemento con el ID especificado. Puedes acceder a sus propiedades y modificarlas según sea necesario.

Class Name


Si deseas cambiar un conjunto de elementos que comparten un nombre de clase, puedes utilizar el método getElementsByClassName(). Este método devuelve una lista de todos los elementos con la clase especificada. Puedes iterar sobre la lista y modificar las propiedades o contenido de los elementos según tus necesidades.

¿Cuál es la sintaxis de la programación en JS?

JavaScript es capáz de distingur entre mayusculas y minusculas, por lo que al declarar dos variables con el mismo nombre pero uno con mayuscula, el lenguaje lo reconocera como dos diferentes.

En JavaScript, las instrucciones o declaraciones se separan con (;), pero no es necesario separar una instruccion con (;) si esta escrita en su propia linea

El texto de JavaScript se escanea de izquierda a derecha

La sintaxis es la misma que en C+

¿Como se declaran variables en JS?

Hay tres formas de declarar una variable, no es necesario mencionar el tipo de dato que es: 'var', 'let', y 'const'

'var': Las variables declaradas con var tienen un alcance de función o global.

'let': Una variable declarada con let estará disponible solo dentro del bloque en el que se declara.

'const': se utiliza para declarar variables constantes, cuyo valor inicial no puede ser reasignado. Las variables declaradas con const también tienen alcance de bloque.

¿Como se declaran funciones en JS?

Para declarar una funcion, se utiliza la palabra 'function', seguido del nombre que llevará la funcion, y un parentesis donde se indican los parametros que utilizara la funcion.

¿Como se llaman funciones en JS?

Simplemente se escribe el nombre de la funcion, y un parentesis que contendran los argumentos que se usaron.

¿Cómo declaras una función interna de un HTML con JS y como declaras un documento externo con JS?

Para declarar una funcion interna, se usa dentro del <head> la etiqueta <script>, dentro de esa etiqueta, se declara la funcion con la sintaxis de JavaScript. posterior, en el body, cuando se utilize la etiqueta button, se utilizara el atributo onclick, donde se debera igualar al nombre de la funcion junto sus atributos.

Para declarar un documento externo, dentro de la etiqueta <script>, src se debe de igualar al nombre del archivo. De igual forma, dentro de la etiqueta de boton, con el atributo onclick se menciona el nombre de la funcion.
